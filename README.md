# Wilson — 中文口播文案转英文语音

**Wilson** 是一款面向短视频运营团队的文本转语音工具。输入中文口播文案，Wilson 自动翻译成地道英文，生成自然、有节奏感的配音音频，支持多音色实时对比和逐句同步高亮。

**无需登录，打开即用。** 所有生成记录保存在共享数据库中，团队成员可随时查看历史工作。

---

## 使用流程

```
中文口播文案 → AI 翻译 → 英文脚本 → 多音色语音生成 → 试听对比 → 下载 MP3
```

全程只需几秒钟。

---

## 核心功能

### 一、语音自然度优化

**问题：** 大多数文本转语音工具听起来像机器人在念书，完全没有口播的感觉。

**Wilson 的做法：**

Wilson 不只是把文字"读"出来。在送入语音引擎之前，会先做一次**智能断句和节奏编排**：

1. **AI 翻译 + 分段**：DeepSeek 大模型将中文口播文案翻译成英文后，自动把长段落拆成适合朗读的短句（每段 4~18 个词）。
2. **智能停顿**：每段之间插入精确控制的停顿，模拟真人口播的自然呼吸感：
   - **0.18 秒** — 极短气口，语句间轻微停顿
   - **0.25 秒** — 短语过渡，冒号、破折号后的停顿
   - **0.40 秒** — 正常句末停顿
   - **0.45 秒** — 钩子提问后的悬念停顿
   - **0.55 秒** — 结尾号召语的冲击停顿
3. **语音参数调优**：使用 ElevenLabs `eleven_multilingual_v2` 模型，配合专为短视频口播调校的参数，让声音有活力、有情绪，而不是死板朗读。

最终生成的音频是一段**有节奏、有呼吸、有情绪**的英文口播，不是机械地把单词串起来念。

---

### 二、多音色并发对比

**问题：** 不知道哪个音色最适合自己的内容，一个个试太慢。

**Wilson 的做法：**

同一段英文文案，Wilson **同时调用三种不同音色**并行生成：

| 音色 | 特点 | 适合场景 |
|------|------|----------|
| **Sarah** | 女声，柔和清晰 | 温柔推荐、知识分享 |
| **Adam** | 男声，低沉自然 | 权威讲解、沉稳口播 |
| **Antoni** | 男声，饱满有张力 |  energetic 内容、产品推荐 |

三种声音的音频并排展示，你可以：
- **独立试听** — 点哪个听哪个
- **自动互斥** — 播放一个时其他自动暂停，不会混音
- **选定下载** — 必须选中一个音色才能下载对应 MP3

生成一次，就能快速对比出最适合的音色。

---

### 三、逐句同步高亮

**问题：** 音频和文字对不上，听的时候不知道读到哪句了。

**Wilson 的做法：**

Wilson 使用 ElevenLabs 的**带时间戳生成接口**，每条音频都会附带精确的字符级时间数据。后端将数据映射到句子级别，前端播放时通过 `requestAnimationFrame` 实时追踪播放进度，让当前正在朗读的句子**高亮显示**。

这意味着：
- 音频播到第一句，第一句亮
- 音频播到第二句，第二句亮
- 三种音色的时间数据各自独立，切换声音不会错乱
- 暂停后恢复，高亮位置自动续上

这是基于**真实时间戳数据**的精确同步，不是粗略的百分比估算。

---

### 四、同一任务二次修改

**问题：** AI 翻译结果不完美，但重新生成要再从中文翻译一遍。

**Wilson 的做法：**

翻译结果默认是**只读模式**。如果觉得英文有瑕疵：

1. 点击 **Edit** 按钮 → 进入编辑模式
2. 直接在文本框里修改英文文案
3. 点击 **Regenerate Voices** → Wilson **跳过翻译**，直接用修改后的英文重新生成三种音色的音频
4. 高亮和下载功能同步更新

翻译一次，调整多次，每次都能快速听到新效果。

## 其他特性

### 无需登录

打开网页就能用。没有注册、没有登录、没有任何门槛。

### 共享历史记录

所有生成的内容保存在共享数据库中。左侧边栏的 **Recent** 显示全部历史记录，任何人打开 Wilson 都能看到。这既是个人工具，也是团队协作的公共工作台。

### New Chat

点击左侧边栏的 **New chat**，回到首页开始全新生成。

---

## 技术栈

- **前端：** React + TypeScript + Tailwind CSS
- **后端：** Hono + tRPC
- **数据库：** MySQL（共享持久化存储）
- **翻译：** DeepSeek（通过 OpenRouter）
- **语音合成：** ElevenLabs API（带时间戳生成）
- **部署：** Kimi 平台
